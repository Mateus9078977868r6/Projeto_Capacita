    Estudante de Ciência da Computação no 2º Período e aspirante a futuro programador, meu objetivo é me desenvolver no mercado da tecnologia, com um foco especial no desenvolvimento front-end. Atualmente, dedico meus estudos iniciais para dominar a base da construção web. A aula de hoje é sobre tags HTML, a estrutura fundamental de qualquer página. Estou aprendendo a utilizar elementos como (títulos),(parágrafos),(links) e (imagens) para dar forma e conteúdo a este que é meu primeiro projeto. Esta é a minha página inicial do site, onde aplico os conceitos teóricos na prática, construindo a interface do zero. No entanto, minha jornada não para por aqui. A longo prazo, busco aprofundar meus conhecimentos em:

  • Estilização e Design: Dominar CSS para criar interfaces visualmente atraentes e responsivas.
  • Interatividade: Aprender JavaScript para adicionar lógica e dinamismo às aplicações web.
  • Bibliotecas e Frameworks: Explorar ferramentas modernas como React ou Vue.js para construir interfaces de usuário complexas e eficientes.
